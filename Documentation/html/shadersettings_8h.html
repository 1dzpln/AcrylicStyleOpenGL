Режимы смешивания.

Enumerator
CM_ADD 

Сложение с учетом прозрачности: C=alfa*A+(1-alfa)*B.

CM_EXCLUSION 

Исключение с последующим учетом прозрачности C=A+B-2*(A*B) -> D=alfa*C+(1-alfa)*B.

CM_SCREEN 

Экран с последующим учетом прозрачности C=A+B-(A*B) -> D=alfa*C+(1-alfa)*B.

CM_MULTIPLY 

Умножение с последующим учетом прозрачности C=A*B -> D=alfa*C+(1-alfa)*B.

CM_LIGHTEN 

Замена светлым с последующим учетом прозрачности max(A*B) -> D=alfa*C+(1-alfa)*B.
